VEX 编程语言 v2.0（十六进制恶魔语言）

VEX（十六进制恶魔语言）v2.0 是我独立开发的一款完整的自定义编程语言，基于字符-十六进制映射核心特性，新增了变量、运算、条件判断、循环等编程语言基础功能，从简单的编解码工具升级为可编写完整程序的自定义语言。

版本更新日志

  • v1.0 - 基础功能：字符串↔十六进制编码双向转换
  • v1.1 - 新增功能：文件批量编解码 + 专属Logo + 版本号管理
  • v2.0 - 核心升级：变量系统、运算表达式、条件判断、循环语句、增强输出

核心特性

  • 基于ASCII字符集的专属十六进制编码映射规则（自动编解码）
  • 完整的变量系统（支持字符串、数值类型，自动编码字符串）
  • 支持加减乘除、比较运算等数学表达式解析
  • if-else条件判断语句，支持复杂逻辑判断
  • for循环语句，支持自定义步长和循环变量
  • 增强的print输出功能（支持变量、编码/解码双模式输出）
  • 支持文本文件批量编解码，保留原有核心功能

基础语法 - 变量系统

变量定义与管理
// 定义数值变量（整数/浮点数）
set num = 10
set pi = 3.14
set score = 95.5

// 定义字符串变量（自动编码存储）
set name = "VEX Language v2.0"
set msg = 'Hello World!'

// 查看所有已定义变量
vars

// 清空所有变量
clear

// 变量示例输出
print $name  // 输出变量值（自动解码显示）
print $num   // 输出数值变量

基础语法 - 运算表达式

数学运算与比较
// 基础数学运算
calc 10 + 5          // 加法 → 15
calc 20 * 3 - 8      // 乘减 → 52
calc 100 / 4         // 除法 → 25
calc 7 % 3           // 取模 → 1

// 变量运算
set a = 8
set b = 2
calc a + b           // 变量加法 → 10
calc a * b           // 变量乘法 → 16
calc (a + b) * 2     // 复合运算 → 20

// 比较运算（返回布尔值）
calc a > b           // 大于 → True
calc a == b          // 等于 → False
calc a >= 8          // 大于等于 → True
calc b < 5           // 小于 → True

基础语法 - 条件判断

if-else条件语句
// 简单条件判断
set score = 85
if score >= 60 {
    print "考试及格"
}

// 完整if-else判断
set age = 18
if age >= 18 { 
    print "已成年"
    calc age * 2       // 条件成立时执行运算
} else { 
    print "未成年"
}

// 复杂条件判断
set num = 15
if num > 0 && num < 10 {
    print "数字在0-10之间"
} else if num >=10 && num <20 {
    print "数字在10-20之间"
} else {
    print "数字超出范围"
}

基础语法 - 循环语句

for循环（支持自定义步长）
// 基础循环：i从1到5，步长1
for i in 1,5,1 {
    print $i                // 输出循环变量
    calc $i * 10            // 循环内运算
}

// 变量循环：使用变量定义循环范围
set max = 3
for j in 0,$max,1 {
    print "循环次数：$j"
}

// 自定义步长循环：从0到10，步长2
for k in 0,10,2 {
    print "步长2循环：$k"
    calc $k + 5
}

完整程序示例

VEX v2.0 完整示例程序
// VEX语言v2.0 完整示例
// 1. 定义变量
set app_name = "VEX 十六进制恶魔语言"
set version = 2.0
set max_count = 3

// 2. 打印程序信息
print $app_name
print "当前版本：$version"

// 3. 条件判断版本号
if version >= 2.0 {
    print "✅ 这是最新版本"
    calc version * 10  // 版本号运算
} else {
    print "⚠️  需要升级版本"
}

// 4. 循环计数并输出
for count in 1,$max_count,1 {
    print "当前计数：$count"
    calc $count * 2    // 循环内运算
}

// 5. 查看所有变量
vars

// 执行结果预览：

// ✅ 条件成立：version >= 2.0 = True
// 📤 [数值输出] 20.0
// 🔄 开始循环：count 从 1 到 3，步长 1
// 📤 [数值输出] 1
// 📤 [数值输出] 2
// 📤 [数值输出] 2
// 📤 [数值输出] 4
// 📤 [数值输出] 3
// 📤 [数值输出] 6
// 📋 当前变量列表：
//   app_name: VEX 十六进制恶魔语言 (编码：56 45 58 20 31 36 进制恶魔语言)

//   count: 3
